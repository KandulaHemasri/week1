This project is a high-performance, production-grade landing page for "Prodesk IT". It goes beyond the standard requirements by implementing a **Fixed Ambient Background**, **Expanded Service Grid**, and **Trust Signals** (Stats Section).

**Difficulty Level:** Level 3 (Advanced) + Bonus UI Polish

##  Key Features
* **Ambient Depth Effect :** A fixed, animated background with moving color blobs that provides a continuous immersive experience across the entire site.
* **Dark Mode :** Fully functional theme toggle with LocalStorage persistence and vibrant blending modes for the background.
* **Glassmorphism :** Custom CSS backdrop-filters applied to the sticky navbar and content cards for a modern, frosted glass aesthetic.
* **Scroll Animations :** Intersection Observer API triggers fade-in and slide-up animations as the user scrolls.
* **Expanded Content :** A full 6-card service grid and a "Why Choose Us" statistics section with animated counters.
* **Responsive Architecture :** Mobile-first design with a custom animated hamburger menu and slide-down dropdown.
* **Semantic HTML:** Optimized for Accessibility (A11y) and SEO.

##  Tech Stack
* **HTML5** (Semantic Structure)
* **Tailwind CSS** (via CDN for development speed)
* **Vanilla JavaScript** (DOM manipulation, LocalStorage, Intersection Observer)
